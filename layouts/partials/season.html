{{- /* Resolve the Season landing Page for any Survival page. Prefers the `season` front-matter param (flat category articles carry `season: N`); falls back to a "Season N" section ancestor for pages that are still season-nested (shops, public-resources, tunnels, map). Returns the season Page, or an empty string when there is none. */ -}} {{- $season := "" -}} {{- with .Params.season -}} {{- $season = site.GetPage (printf "/survival/season-%v" .) -}} {{- else -}} {{- range .Ancestors -}} {{- if and (not $season) (hasPrefix .LinkTitle "Season") -}}{{- $season = . -}}{{- end -}} {{- end -}} {{- end -}} {{- return $season -}}
