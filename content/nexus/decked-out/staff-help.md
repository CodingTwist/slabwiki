If a run goes wrong, these commands get you out:

- `/stuck` - emergency escape. **Causes death**, so use it only when you're
  genuinely trapped.
- `/leave` - the final escape option. It **voids your rewards**, but staff can
  roll the run back if something broke through no fault of yours.

If the map corrupts (usually from swapping co-op colours mid-session) or the
system fails to hand out rewards it should have, contact staff - most run
problems can be rolled back.
